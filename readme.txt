Preinstallion is Gurobi (https://www.gurobi.com/). This work is done by version 9.1.

This repository contains the codes for generating the results in Tab. 4. In the current codes, the input pattern (|H_{_}|,|H_x|,|H_s|) is set to class (2,1,0).

To get the 4 round results, run

```
make 2round_optimal_enum
```

To get the 5 round results, run

```
make 3round_optimal_enum
```

To get the 4 round results, run

```
make 4round_optimal_enum
```
